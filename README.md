# Typst Template
This is my term paper/thesis template for typst.

## Requirements
- Typst
- VSCode with Typst LSP (optional)
- Internet Access to download packages  
If you are behind a proxy, you need to tell typst about it:  
`export https_proxy=...`

## Building
How to compile:
```shell
typst compile --root . src/main.typ ./out.pdf
```
